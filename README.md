# おこづかいアプリ

このアプリは、ご家庭でお子さんにやってもらいたいお手伝いを登録して、そのお手伝いをお子さんがきちんと実施したら、設定されたお小遣いをお子さんの口座に振り込む、という擬似銀行口座アプリです。

## アプリ使用方法

【ご両親（管理者）側の使用方法】


### 1.ログイン

このアプリには、初めから2つのアカウントが用意されています。

まずユーザー名adminの方のアカウントでログインしましょう。（testの方は権限が低く、管理者メニューが出ません。一応作っておいた程度のものです。）

●アカウント1

ユーザー名：admin

パスワード：password

●アカウント2

ユーザー名：test

パスワード：password

###*2.管理メニューからお手伝い検索*

トップページから「管理メニュー」をクリック→「お手伝い」クリック→「お手伝いリスト」クリック→「検索ボタン」押下。

すると、現在登録されているお手伝い一覧が表示されます。キーワードを入力すれば、それにヒットしたお手伝いだけが表示されます。

このページから、お手伝いの編集や削除ができます。


###*3.管理メニューからお手伝い登録*

（まずお手伝いメニューに戻る必要があります。）

今度は、お手伝いメニュー画面から「お手伝い登録」をクリックしましょう。

すると、お手伝いの名前と、そのお手伝いに対して支払うお小遣いが登録できます。お子さんにやってほしいなというものを登録しましょう。

ここで登録したお手伝いは、「2.管理メニューからお手伝い検索」と同じ方法で検索できます。

###*4.ユーザー登録*

お子さんに使ってほしいので、お子さん用のアカウントを作りましょう。

「管理メニュー」をクリック→「ユーザー登録」クリック。

ユーザー名とパスワードを設定し、ロールを「一般ユーザー」として作りましょう。

（管理者権限を与えてしまうと、お子さん自身が勝手にお小遣い振り込めるようになってしまうので気を付けてください）

###*5.口座登録*

お子さん用の口座を作りましょう。

「管理メニュー」をクリック→「口座登録」クリック。

「4.ユーザー登録」で設定したお子さんのユーザー名が「ユーザーID」です。

お子さんの口座に最初から入金しておく金額を「所持金」として登録してください。

ここまできたら、「4.ユーザー登録」で作ったアカウントをお子さんに伝えて、アプリを使ってもらってください。

パスワードは後からお子さん自身が変更できます。

###*6.お手伝いの承認・差し戻し*

「管理メニュー」をクリック→「報告一覧」クリック→「検索ボタン」押下。

すると、お子さんから報告されたお手伝い一覧が表示されます。

そのお手伝いを確かにお子さんがやったと思うなら「承認」ボタンを押します。そうするとお子さんの口座にお小遣いが振り込まれます。

お手伝いをやったと認められないなら、「差戻」ボタンを押してください。そのお手伝いはお子さんに差し戻されます。


【お子さん側の使用方法】

*1.ログイン*

ご両親から教えてもらったアカウントでログインしましょう。

*2.お手伝いをMyリストへ登録*

トップページから「ユーザーメニュー」をクリック→「お手伝い」ボタンクリック→「お手伝い」クリック→「検索ボタン」押下。

お手伝いの一覧が表示されます。この中から、自分がやりたいお手伝いについて「登録」ボタンを押してMyリストに登録します。

*3.Myリストからお手伝い検索・登録*

（まずお手伝いメニューに戻る必要があります。）

今度は、お手伝いメニュー画面から「Myリスト」をクリックしましょう。

「検索」ボタンを押すと、「2.お手伝いをMyリストへ登録」で登録したお手伝いが出てきます。

このお手伝いをしたなら、「報告」ボタンを押してご両親に報告します。（認められればお小遣いが振り込まれます。）

やっぱりやりたくないな、と思ったら「削除」ボタンを押しましょう。

  *4.差し戻し一覧の確認*
  
（まずお手伝いメニューに戻る必要があります。）

今度は、お手伝いメニュー画面から「差し戻し」をクリック→「検索ボタン」押下。

Myリストから、お手伝いをした報告をしたけれども、ご両親から認められず差し戻される時があります。

そのとき、差し戻されたお手伝いが表示されるのがこのページです。

あらためてお手伝いをして、「報告」ボタンを押し、今度はご両親に認めてもらえるよう頑張りましょう。

やっぱりやりたくないな、と思ったら「取下げ」ボタンを押しましょう。
